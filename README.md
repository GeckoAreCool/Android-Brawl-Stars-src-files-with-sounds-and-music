# Android-Brawl-Stars-src-files-with-sounds-and-music
Src Brawl stars(Android) with code, music, sounds, images

Download and watch.

Test source:
1. Download apktool m on your android device(or Android emulator) here > https://trashbox.ru/files30/2465940/apktool_m_v2.4.0-2603282026032801.apk/ .
2. Share zip package with your phone or install directly from this site from an your android device(or Android emulator).
3. Go to File Explorer on your Android device(or Android emulator) and unzip the zip package.
4. Go to apktool m, then go to the folder where you unzipped the zip package and click the button at the top (build project).
5. Once the project is installed, you can test the source just click to Brawl Stars.
